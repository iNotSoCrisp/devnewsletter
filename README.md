# LastScroll

A dev newsletter platform — built as a monorepo.

## Structure

```
lastscroll/
├── frontend/   # React + Vite — newsletter landing page & web app
├── backend/    # API server (coming soon)
└── package.json
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### From root (after backend is set up)
```bash
npm run dev      # runs frontend dev server
npm run build    # builds frontend
npm run start    # starts backend server
```
